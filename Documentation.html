Window Summary

There are two windows in the Grapher software. The Graph shows the functions graphed on a cartesian plane. The Control Window contains the controls for the Graph

Graph

The Graph shows the functions graphed on a cartesian plane.

The dark black lines on the graph are the x and y axes. The lighter gray lines are the unit lines. The interval between the unit lines are set under the Window Controls menu. The colored lines are the functions.

Function menu

The function menu contains the function controls

The Graph can contain up to ten functions. The variable "x" is automatically incremented to create the graph. Variables and sub-functions("x', constants, sin(), etc) do not assume multiplication when placed next to another term.
"6x + 1" is invalid, "6*x + 1" is valid
"2(4)" is invalid, "2*(4)" is valid
There several built in sub-functions:
sin() returns the sine of a given angle(in radians)
cos() returns the cosine of a given angle(in radians)
tan() returns the tangent of a given angle(in radians)
asin() returns the angle(in radians) of a given sine
acos() returns the angle(in radians) of a given cosine
atan() returns the angle(in radians) of a given tangent
sqrt() returns the square root of a given number
exp() returns euler's number e raised to a given number
ln() returns the Natural Logarithm of a given number
log() returns the Common Logarithm of a given number
log2() returns the logarithm base 2 or a given number

Pressing "Update Functions" will regraph the given functions

Constant Menu

The Constant Menu keeps track of constants used in the functions.

The Name Field is the name of the constant. The value field is the value of the constant.
Clicking "Add" will add the constant to the list of constants to be evaluated in the functions. Clicking the red "X" next to a listed constant will delete that constant. Adding a constant with the same name of an existing constant will overwrite the original constant's value to the new constant's value.
To use a constant in a function simply insert the given name into the equation.

Window Menu

The Window Menu controls the bounds of what is visible in the Graph

The Window Menu contains 7 fields:
1)The x lower bound is the lowest x value visible on the graph
2)The x upper bound is the highest x value visible on the graph
3)The y lower bound is the lowest y value visible on the graph
4)The y upper bound is the highest y value visible on the graph
5)The x interval sets the distance between the light gray interval lines on the x axis
6)The y interval sets the distance between the light gray interval lines on the y axis
7)The definition sets the preciseness of the graph. A smaller definition will be more accurate, a larger definition will be less accurate.

Once the fields have been properly filled out, pressing "Update" will set the visible window in the Graph
